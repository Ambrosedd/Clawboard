# Clawboard 交互原型

## 当前版本
Prototype V3（可点击交互版）

## 文件
- `index.html`
- `styles.css`
- `app.js`
- `server.js`
- `prototype-screenshot-desktop.png`
- `prototype-screenshot-mobile.png`

## 打开方式
直接用浏览器打开 `index.html`，或启动本地静态服务器后访问。

## 当前支持
- 页面切换
- Tab 切换
- 审批操作反馈
- 任务控制反馈
- 模拟配对成功
- 筛选状态切换

## 用途
这个原型用于：
- 展示 App First 的产品形态
- 验证主要页面结构和核心操作路径
- 为 SwiftUI 正式开发提供视觉与交互参考
