# 龙虾 OS App First 静态原型

## 文件
- `index.html`
- `styles.css`
- `server.js`
- `prototype-screenshot-desktop.png`
- `prototype-screenshot-mobile.png`

## 打开方式
直接用浏览器打开 `index.html` 即可。

## 设计目标
- 简洁美观
- App First
- 本地优先
- 突出移动端控制、审批、查看进度

## 当前展示内容
- 首页 Dashboard
- 龙虾列表
- 任务详情
- 审批中心
- 设置/连接引导

## 说明
这是静态原型，主要用于确定视觉风格和信息结构，不包含真实交互逻辑。
