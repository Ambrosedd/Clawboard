# Clawboard 原型交互说明

## 当前版本
- 版本：Prototype V3
- 目标：从静态展示原型升级为可点击演示原型

## 已实现交互

### 1. 页面切换
支持通过以下方式切换页面：
- 左侧导航
- 手机底部 Tab
- 页面中的跳转按钮

可切换页面：
- 首页
- 龙虾
- 任务详情
- 审批中心
- 设置

### 2. 审批交互
支持模拟：
- 批准
- 拒绝
- 缩小范围（当前以反馈形式模拟）

点击“批准”后会产生：
- 待审批数减少
- 任务进度从 72% 提升到 84%
- 页面顶部浮动提示反馈

### 3. 任务控制交互
支持模拟：
- 暂停
- 终止

当前以浮动提示的形式反馈操作结果。

### 4. 配对交互
设置页提供：
- 模拟扫码配对成功

点击后会更新：
- 配对状态文案
- 浮动操作提示

### 5. 筛选交互
龙虾页支持点击筛选项：
- 全部
- 运行中
- 待审批
- 异常

当前先实现为视觉切换与反馈提示，后续可以进一步扩展为真实列表过滤。
